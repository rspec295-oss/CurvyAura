# How to Upload Images to GitHub

Uploading images to GitHub is a fairly straightforward process. Follow these detailed step-by-step instructions to upload images effectively.

## Step 1: Navigate to Your Repository
1. Log in to your GitHub account.
2. Go to your repository where you want to upload the image. For example, `rspec295-oss/CurvyAura`.

## Step 2: Create/Create a New Folder (Optional)
1. If you want to organize your images, consider creating a new folder. Click on the `Add file` button at the top right, and select `Create new folder`.
2. Name the folder (e.g., `images`), and click `Create folder`.

## Step 3: Upload Images
1. Inside your repository or the desired folder, click on the `Add file` button and select `Upload files`.
2. Drag and drop your image files into the upload area or click `choose your files` to open a file browser to select images. You can upload multiple files at once.

## Step 4: Commit the Changes
1. Once your images are uploaded, scroll down to the `Commit changes` section.
2. Enter a commit message (e.g., `Upload images`).
3. Choose whether to commit directly to `main` or create a new branch for this upload.
4. Click `Commit changes`.

## Step 5: Verify Upload
1. You should now see your uploaded images in the repository.
2. Click on the image file to view it and ensure it uploaded correctly.

## Tips:
- Make sure your image files are in supported formats (PNG, JPG, GIF, etc.).
- Keep your images appropriately sized for web use to optimize loading times.

That's it! You've successfully uploaded images to GitHub.
